# MakersArena - Website

1.  Clone the repository:

        git clone https://github.com/MakersArena/website.git
1. Change directory:
  
       cd website
1.  Install dependencies using npm:

        npm install
1.  Start the app:

        npm start
1.  View the app at [http://localhost:8080](http://localhost:8080).

1.  Stop the app by pressing `Ctrl+C`.
